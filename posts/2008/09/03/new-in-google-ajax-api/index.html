Новенькое в Google AJAX API 03.09.2008

Теперь можно загружать любой апи по желанию на любом этапе работы скриптов: Dynamic Loading. Раньше можно было только в head, на этапе загрузки (если не подхачить маленько). Даже загрузку самого загрузчика можно отложить.

А еще появилась возможность узнать географическое расположение клиента. Приблизительно, конечно, но зато просто, быстро и бесплатно.
