Не хотите ставить meta http-equiv="imagetoolbar" content="no" на всех страницах? Не срабатывает set_header? Тогда поможет экспрешн.


img
{
  scrollbar-highlight-color: expression
  (
    (runtimeStyle.scrollbarHighlightColor = "transparent"),
    (this.galleryImg='no')
  )
}

Ключевой момент — это this.galleryImg='no'. По строкам я разбил для красоты, перед использованием их надо объединить в одну.
